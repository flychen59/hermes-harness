# 设计系统：Warp

## 1. 视觉主题与氛围

Warp 的网站宛如坐在深林中的篝火旁——温暖、深沉，充满静谧的自信。与大多数开发工具偏爱的冷色调蓝黑不同，Warp 用温暖的近黑色包裹一切，如同焦木或深色土壤。文字也不是纯白——而是 Warm Parchment（`#faf9f6`），一种几乎不可察觉的奶油色，柔化了每个标题，让深色画布显得亲切而非冷峻。

字体是 secret 利器：Matter，一款几何无衬线字体，具有独特的个性，几乎在所有文字中都使用 Regular 字重。这种字体选择对于开发工具来说非同寻常——Matter 具有一种柔和感和人情味，传递出"这个终端适合所有人，不仅仅是老派程序员"的信号。结合紧凑的行高和标题上控制的负字间距，效果既精致又平易近人。自然摄影穿插在终端截图之间，创造了一种视觉语言：这款工具带你进入心流，进入平静高效的状态。

整体设计理念是通过温暖来实现克制。极少的色彩（几乎是单色的暖灰色）、极少的装饰，专注于产品展示，背景是电影感的深色风景。这是一家像生活方式品牌一样营销的终端公司。

**关键特征：**
- 温暖的深色背景——不是冷黑色，而是带有暖灰色基调的泥土感近黑色
- Warm Parchment（`#faf9f6`）文字而非纯白——微妙的奶油温暖感
- Matter 字体家族（Regular 字重）——几何感但平易近人，不是典型的开发工具字体
- 自然摄影与产品截图交织——生活方式与开发工具的融合
- 几乎单色的暖灰色调色板——没有大胆的强调色
- 大写标签配宽字间距（2.4px）用于分类——编辑风格标识
- 药丸形深色按钮（`#353534`，50px 圆角）——克制的、柔和的 CTA

## 2. 色彩系统与角色

### 主色
- **Warm Parchment**（`#faf9f6`）：主要文字颜色——一种近乎奶油色的米白，柔化每个表面
- **Earth Gray**（`#353534`）：按钮背景、深色交互表面——温暖而非冰冷
- **Deep Void**（近黑色，页面背景）：源自 body 背景的温暖深色画布

### 辅助色与强调色
- **Stone Gray**（`#868584`）：次要文字、柔和描述——温暖的中灰色
- **Ash Gray**（`#afaeac`）：正文、按钮文字——常用的阅读色
- **Purple-Tint Gray**（`#666469`）：带有微妙紫色基调的链接文字——内容中带下划线的链接

### 表面与背景
- **Frosted Veil**（`rgba(255, 255, 255, 0.04)`）：超微妙的白色叠加，用于表面区分
- **Mist Border**（`rgba(226, 226, 226, 0.35)` / `rgba(227, 227, 227, 0.337)`）：半透明边框，用于卡片边界
- **Translucent Parchment**（`rgba(250, 249, 246, 0.9)`）：略微透明的主要表面，营造深度感

### 中性色与文字
- **Warm Parchment**（`#faf9f6`）：标题、高强调文字
- **Ash Gray**（`#afaeac`）：正文段落、描述
- **Stone Gray**（`#868584`）：次要标签、次要信息
- **Muted Purple**（`#666469`）：下划线链接、第三级内容
- **Dark Charcoal**（`#454545` / `#353534`）：边框、按钮背景

### 语义色与强调色
- Warp 采用几乎单色的系统——没有大胆的强调色
- 交互状态通过透明度变化和下划线装饰来传达，而非颜色变化
- 任何强调色都会破坏这个温暖、克制的调色板

### 渐变系统
- 营销网站上没有明确的渐变
- 深度通过分层的半透明表面和摄影来创造，而非颜色渐变

## 3. 排版规则

### 字体家族
- **展示与正文**：`Matter Regular`——几何无衬线字体，具有柔和个性。后备字体：`Matter Regular Placeholder`，系统无衬线字体
- **中等字重**：`Matter Medium`——字重 500 变体，用于强调。后备字体：`Matter Medium Placeholder`
- **方形**：`Matter SQ Regular`——方形变体，用于特定展示场景。后备字体：`Matter SQ Regular Placeholder`
- **UI 辅助**：`Inter`——用于特定 UI 元素。后备字体：`Inter Placeholder`
- **等宽展示**：`Geist Mono`——用于代码/终端展示标题
- **等宽正文**：`Matter Mono Regular`——自定义等宽伴侣字体。后备字体：`Matter Mono Regular Placeholder`

### 原则
- **Regular 字重主导**：几乎所有文字都使用字重 400（Regular）——甚至标题。Matter Medium（500）仅用于卡片标题和按钮等强调时刻。这创造了一种非常均匀、平静的排版质感
- **大写作为编辑信号**：小标签和分类使用大写转换配宽字间距（1.4px–2.4px），创造杂志编辑风格的分类系统
- **温暖的易读性**：Matter 的几何柔和感 + 温暖的文字颜色（#faf9f6）+ 控制的负字间距的组合，创造了在深色表面上轻松阅读、富有人情味的文字
- **无粗体展示**：完全不使用粗体（700+）字重——克制是核心理念

## 4. 组件样式

### 按钮
- **深色药丸按钮**：`#353534` 背景，Ash Gray（`#afaeac`）文字，药丸形状（50px 圆角），`10px` 内边距。主要的 CTA——温暖、柔和、低调
- **磨砂标签**：`rgba(255, 255, 255, 0.16)` 背景，黑色文字（`rgb(0, 0, 0)`），矩形（6px 圆角），`1px 6px` 内边距。小型行内标签式按钮
- **幽灵按钮**：无可见背景，仅文字，悬停时显示下划线装饰
- **悬停**：微妙的透明度或亮度变化——没有戏剧性的颜色改变

### 卡片与容器
- **摄影卡片**：全出血自然图像配叠加文字，8px–12px 圆角
- **终端截图卡片**：产品 UI 嵌入在带圆角（8px–12px）的深色容器中
- **带边框卡片**：半透明边框（`rgba(226, 226, 226, 0.35)`）用于边界，12px–14px 圆角
- **悬停**：最小化——内容卡片悬停时不会剧烈变化，保持平静的美学

### 输入框与表单
- 营销网站上表单存在感极小
- 深色背景输入框配暖灰色文字
- 焦点状态：边框亮度增加，无彩色光环（与单色调色板一致）

### 导航
- **顶部导航**：深色背景，暖米色品牌文字，Matter Regular 16px 用于链接
- **链接颜色**：Stone Gray（`#868584`）用于柔和导航，Warm Parchment 用于激活/悬停状态
- **CTA 按钮**：导航末端的深色药丸按钮（#353534）——克制，不引人注目
- **移动端**：折叠为简化导航
- **粘性**：导航滚动时固定

### 图片处理
- **自然摄影**：风景、森林、黄金时刻场景——对于开发工具来说完全独特
- **终端截图**：产品 UI 显示在真实的终端窗口框架中
- **混合构图**：自然图像和终端截图穿插，创造生活方式与工具结合的叙事
- **全出血**：图片通常跨越整个容器宽度，8px 圆角
- **视频**：视频元素带有 10px 圆角

### 推荐区域
- 社会证明区域（"Don't take our word for it"）配引言
- 柔和样式与整体克制一致

## 5. 布局原则

### 间距系统
- **基础单位**：8px
- **比例**：1px, 4px, 5px, 8px, 10px, 12px, 14px, 15px, 16px, 18px, 24px, 26px, 30px, 32px, 36px
- **区块内边距**：主要区块之间 80px–120px 垂直间距
- **卡片内边距**：16px–32px 内部间距
- **组件间距**：相关元素之间 8px–16px

### 网格与容器
- **最大宽度**：约 1500px 容器（断点在 1500px），居中
- **列模式**：全宽主视觉、双列功能区块配摄影、单列推荐
- **电影感布局**：宽容器让摄影有呼吸空间

### 留白哲学
- **广阔而温暖**：区块之间慷慨的间距——深色背景创造了温暖的虚空，感觉是沉思的而非空洞的
- **摄影作为留白**：自然图像在密集的产品信息之间作为视觉呼吸空间
- **编辑节奏**：布局读起来像一本杂志——每个区块都是精心设计的翻页时刻

### 圆角比例
- **4px**：小型交互元素——按钮、标签
- **5px–6px**：标准组件——链接、小型容器
- **8px**：图片、视频容器、标准卡片
- **10px**：视频元素、中型容器
- **12px**：功能卡片、大图片
- **14px**：大型容器、突出卡片
- **40px**：大圆角区块
- **50px**：药丸按钮——主要 CTA
- **200px**：进度条——完整药丸形状

## 6. 深度与层级

| 层级 | 处理方式 | 用途 |
|------|----------|------|
| 层级 0（平坦） | 无阴影，深色背景 | 页面画布、大多数表面 |
| 层级 1（薄纱） | `rgba(255, 255, 255, 0.04)` 叠加 | 微妙的表面区分 |
| 层级 2（边框） | `rgba(226, 226, 226, 0.35) 1px` 边框 | 卡片边界、区块分隔 |
| 层级 3（环境） | `rgba(0, 0, 0, 0.2) 0px 5px 15px`（从设计推断） | 图片容器、浮动元素 |

### 阴影哲学
Warp 的层级系统非常平坦——营销网站上几乎没有阴影使用。深度通过以下方式传达：
- **半透明边框**代替阴影——35% 透明度的边框创造出幽灵般的边界感
- **摄影分层**——图像创造自然深度，无需人工阴影
- **表面透明度变化**——`rgba(255, 255, 255, 0.04)` 叠加创造出几乎不可察觉的层级差异
- 效果平静而扎实——没有东西漂浮，一切都在安放

### 装饰性深度
- **摄影作为深度**：自然图像创造出阴影无法实现的氛围深度
- **无毛玻璃或模糊效果**：设计完全避免时髦的 glassmorphism
- **温暖的环境感**：任何光晕都来自摄影的自然光线，而非人工 CSS

## 7. 做与不做

### 做
- 使用温暖的米白色（`#faf9f6`）作为文字而非纯白——奶油基调是关键
- 保持按钮克制和柔和——深色填充（#353534）配柔和文字（#afaeac），无鲜艳 CTA
- 几乎所有内容都使用 Matter Regular（字重 400）——甚至标题。仅保留 Medium（500）用于强调
- 使用大写标签配宽字间距（1.4px–2.4px）进行分类
- 将自然摄影与产品截图穿插——这是品牌标识的核心
- 保持几乎单色的暖灰色调色板——无大胆强调色
- 使用半透明边框（`rgba(226, 226, 226, 0.35)`）代替阴影作为卡片边界
- 标题保持负字间距（-0.4px 至 -2.4px），实现 Matter 的压缩展示效果

### 不做
- 使用纯白（#ffffff）作为文字——始终是 warm parchment（#faf9f6）
- 添加大胆强调色（蓝、红、绿）——系统刻意采用单色暖灰色
- 对任何文字应用粗体（700+）——Warp 从不超过 Medium（500）
- 使用浓重的投影——深度来自边框、摄影和透明度变化
- 创建冷色调或蓝色调的深色背景——温暖是关键
- 添加装饰性渐变或光晕效果——摄影提供所有视觉趣味
- 使用紧凑、压缩的布局——编辑风格间距慷慨而沉思
- 混入 Matter 家族 + Inter 辅助以外的额外字体

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|------|----------|
| 移动端 | <810px | 单列、堆叠区块、主视觉文字缩小至约 48px、汉堡导航 |
| 平板 | 810px–1500px | 双列功能区块开始、摄影缩放、导航链接部分可见 |
| 桌面 | >1500px | 完整电影感布局、80px 主视觉展示、摄影与文字并排 |

### 触控目标
- 药丸按钮：50px 圆角配 10px 内边距——舒适的触控目标
- 导航链接：16px 文字配周围内边距，保证可访问性
- 移动端 CTA：移动端全宽药丸按钮，方便拇指触及

### 折叠策略
- **导航**：完整水平导航 → 简化的移动端导航
- **主视觉文字**：80px 展示 → 56px → 48px 跨断点变化
- **功能区块**：摄影与文字并排 → 垂直堆叠
- **摄影**：在容器内响应式缩放，保持电影感宽高比
- **区块间距**：按比例缩减——桌面端慷慨 → 移动端紧凑

### 图片行为
- 自然摄影响应式缩放，保持宽电影感比例
- 终端截图在响应式容器内保持宽高比
- 视频元素缩放时保持 10px 圆角
- 无艺术指导变化——跨断点使用相同构图

### 迭代指南
在使用此设计系统优化现有界面时：
1. 验证文字颜色是 warm parchment（#faf9f6）而非纯白——温暖感微妙但关键
2. 确保所有按钮使用克制的深色调色板（#353534）——无鲜艳或彩色 CTA
3. 检查 Matter Regular（400）是默认字重——Medium（500）仅用于强调
4. 确认大写标签有宽字间距（1.4px–2.4px）——紧凑的大写在这里感觉不对
5. 整体基调应该感觉温暖和平静，像一本精心设计的杂志——而非激进或科技炫目
