# 设计系统：Cursor

## 1. 视觉主题与氛围

Cursor 的网站是温暖极简主义与代码编辑器优雅感的典范。整个体验建立在温暖的米白色画布（`#f2f1ed`）之上，配以深暖棕色的文字（`#26251e`）——不是纯黑，不是中性灰，而是一种带有黄色底色的深邃温暖的近黑色，让人联想到旧纸张、墨水和工艺。这种温暖渗透到每一个表面：背景倾向于奶油色（`#e6e5e0`、`#ebeae5`），边框使用 `oklab` 色彩空间消融为透明的温暖覆盖层，甚至错误状态（`#cf2d56`）也带有温暖感而非冷峻的红色。整体感觉更像是一本高端印刷出版物，而非科技网站。

自定义的 CursorGothic 字体是排版签名——一种哥特式无衬线字体，在展示尺寸下使用激进负字间距（72px 时为 -2.16px），营造出压缩、工程化的感觉。作为次要声音，jjannon 衬线字体（带有 OpenType `"cswh"` 上下文花体替换）为正文和编辑性段落提供了文学性的对比。等宽字体来自 berkeleyMono，这是一款精致的编程字体，将营销网站与 Cursor 作为代码编辑器的核心身份连接起来。这种三字体系统（哥特展示、衬线正文、等宽代码）赋予 Cursor 开发者工具领域中最丰富的排版调色板之一。

边框系统尤为独特——Cursor 使用 `oklab()` 色彩空间定义边框颜色，在不同透明度级别（0.1、0.2、0.55）应用温暖的棕色，创造出有机而非机械的边框感觉。标志性边框颜色 `oklab(0.263084 -0.00230259 0.0124794 / 0.1)` 不是简单的 rgba 值，而是一种感知均匀的颜色，在不同背景下保持视觉一致性。

**核心特征：**
- CursorGothic 配合激进负字间距（72px 时 -2.16px，36px 时 -0.72px）用于压缩展示标题
- jjannon 衬线字体用于正文，启用 OpenType `"cswh"`（上下文花体替换）
- berkeleyMono 用于代码和技术标签
- 温暖的米白色背景（`#f2f1ed`）而非纯白——整个系统向暖色调偏移
- 主文字颜色 `#26251e`（带有黄色底色的温暖近黑色）
- 强调橙色 `#f54e00` 用于品牌高亮和链接
- oklab 空间边框在不同透明度级别，用于感知均匀的边缘处理
- 药丸形状元素配合极端圆角（33.5M px，实际上是完整药丸）
- 8px 基础间距系统，配合精细的子 8px 增量（1.5px、2px、2.5px、3px、4px、5px、6px）

## 2. 色彩调色板与角色

### 主色
- **Cursor Dark**（`#26251e`）：主要文字、标题、深色 UI 表面。一种带有明显黄棕底色的温暖近黑色——系统的定义色。
- **Cursor Cream**（`#f2f1ed`）：页面背景、主要表面。不是白色，而是一种设定整体温暖基调的温暖奶油色。
- **Cursor Light**（`#e6e5e0`）：次要表面、按钮背景、卡片填充。一种略微更暖、略深的奶油色。
- **Pure White**（`#ffffff`）：少量用于最大对比度元素和特定表面高亮。
- **True Black**（`#000000`）：最小化使用，特定代码/控制台场景。

### 强调色
- **Cursor Orange**（`#f54e00`）：品牌强调色，`--color-accent`。一种充满活力的红橙色，用于主要 CTA、活动链接和品牌时刻。温暖而紧迫。
- **Gold**（`#c08532`）：次要强调色，温暖金色用于高级或高亮场景。

### 语义色
- **Error**（`#cf2d56`）：`--color-error`。一种温暖的深红玫瑰色而非冷红色。
- **Success**（`#1f8a65`）：`--color-success`。一种柔和的蓝绿色，向暖色调偏移。

### 时间线 / 功能色
- **Thinking**（`#dfa88f`）：温暖桃色，用于 AI 时间线中的"思考"状态。
- **Grep**（`#9fc9a2`）：柔和的鼠尾草绿，用于搜索/grep 操作。
- **Read**（`#9fbbe0`）：柔和蓝色，用于文件读取操作。
- **Edit**（`#c0a8dd`）：柔和薰衣草色，用于编辑操作。

### 表面色阶
- **Surface 100**（`#f7f7f4`）：最浅的按钮/卡片表面，几乎未着色。
- **Surface 200**（`#f2f1ed`）：主要页面背景。
- **Surface 300**（`#ebeae5`）：按钮默认背景，微妙强调。
- **Surface 400**（`#e6e5e0`）：卡片背景，次要表面。
- **Surface 500**（`#e1e0db`）：第三级按钮背景，更深层强调。

### 边框颜色
- **Border Primary**（`oklab(0.263084 -0.00230259 0.0124794 / 0.1)`）：标准边框，oklab 空间中 10% 的温暖棕色。
- **Border Medium**（`oklab(0.263084 -0.00230259 0.0124794 / 0.2)`）：强调边框，20% 温暖棕色。
- **Border Strong**（`rgba(38, 37, 30, 0.55)`）：强边框，表格分割线。
- **Border Solid**（`#26251e`）：完全不透明深色边框，用于最大对比度。
- **Border Light**（`#f2f1ed`）：浅色边框，与页面背景匹配。

### 阴影与深度
- **Card Shadow**（`rgba(0,0,0,0.14) 0px 28px 70px, rgba(0,0,0,0.1) 0px 14px 32px, oklab(0.263084 -0.00230259 0.0124794 / 0.1) 0px 0px 0px 1px`）：重型悬浮卡片，带有温暖 oklab 边框环。
- **Ambient Shadow**（`rgba(0,0,0,0.02) 0px 0px 16px, rgba(0,0,0,0.008) 0px 0px 8px`）：微妙的环境光晕，用于浮动元素。

## 3. 排版规则

### 字体家族
- **展示/标题**：`CursorGothic`，回退字体：`CursorGothic Fallback, system-ui, Helvetica Neue, Helvetica, Arial`
- **正文/编辑**：`jjannon`，回退字体：`Iowan Old Style, Palatino Linotype, URW Palladio L, P052, ui-serif, Georgia, Cambria, Times New Roman, Times`
- **代码/技术**：`berkeleyMono`，回退字体：`ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, Liberation Mono, Courier New`
- **UI/系统**：`system-ui`，回退字体：`-apple-system, Segoe UI, Helvetica Neue, Arial`
- **图标**：`CursorIcons16`（图标字体，14px 和 12px）
- **OpenType 特性**：jjannon 正文启用 `"cswh"`，CursorGothic 按钮/标题启用 `"ss09"`

### 原则
- **哥特压缩以增强冲击力**：CursorGothic 在展示尺寸使用 -2.16px 字间距（72px），逐渐放宽：36px 时 -0.72px，26px 时 -0.325px，22px 时 -0.11px，16px 及以下为 normal。这种字距处理创造了精密工程的感觉。
- **衬线赋予灵魂**：jjannon 提供了文学的温暖感。`"cswh"` 特性添加了上下文花体替换，赋予正文书法般的品质。
- **三种排版声音**：哥特（展示/UI）、衬线（编辑/正文）、等宽（代码/技术）。每种服务于不同的沟通目的。
- **字重克制**：CursorGothic 几乎只使用 400 字重，依靠大小和字距来建立层级而非字重。system-ui 组件使用 500-700 用于功能性强调。

**次要药丸**

**第三级药丸**

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 精细刻度：1.5px、2px、2.5px、3px、4px、5px、6px（子 8px 用于微调）
- 标准刻度：8px、10px、12px、14px（从提取中得出）
- 扩展刻度（推断）：16px、24px、32px、48px、64px、96px
- 值得注意：精细的子 8px 增量用于精确的图标/文字对齐

### 网格与容器
- 最大内容宽度：约 1200px
- 英雄区：居中单列，顶部 generous 内边距（80-120px）
- 功能区块：2-3 列网格用于卡片和功能
- 全宽区块配温暖奶油色或略深背景
- 侧边栏布局用于文档和设置页面

### 留白哲学
- **温暖的负空间**：奶油色背景意味着留白具有温暖和质感，不像冷白色极简主义。大片空白感觉舒适而非冰冷。
- **压缩文字，开放布局**：CursorGothic 标题的激进负字间距被周围 generous 的边距平衡。文字紧凑，周围空间呼吸。
- **区块变化**：交替的表面色调（奶油色 → 更浅奶油色 → 奶油色）创造微妙的区块区分，无需生硬的边界。

### 边框圆角刻度
- 微型（1.5px）：精细细节元素
- 小型（2px）：行内元素，代码片段
- 中型（3px）：小容器，行内徽章
- 标准（4px）：卡片，图片，紧凑按钮
- 舒适（8px）：主要按钮，卡片，菜单
- 特色（10px）：较大容器，特色卡片
- 完整药丸（33.5M px / 9999px）：药丸按钮，标签，徽章

## 6. 深度与高度

| 级别 | 处理 | 用途 |
|-------|-----------|-----|
| 扁平（Level 0） | 无阴影 | 页面背景，文字块 |
| 边框环（Level 1） | `oklab(0.263 / 0.1) 0px 0px 0px 1px` | 标准卡片/容器边框（温暖 oklab） |
| 边框中等（Level 1b） | `oklab(0.263 / 0.2) 0px 0px 0px 1px` | 强调边框，活动状态 |
| 环境（Level 2） | `rgba(0,0,0,0.02) 0px 0px 16px, rgba(0,0,0,0.008) 0px 0px 8px` | 浮动元素，微妙光晕 |
| 悬浮卡片（Level 3） | `rgba(0,0,0,0.14) 0px 28px 70px, rgba(0,0,0,0.1) 0px 14px 32px, oklab ring` | 模态框，弹出层，悬浮卡片 |
| 聚焦 | `rgba(0,0,0,0.1) 0px 4px 12px` 在按钮聚焦时 | 交互聚焦反馈 |

**阴影哲学**：Cursor 的深度系统围绕两个理念构建。首先，边框使用感知均匀的 oklab 色彩空间而非 rgba，确保温暖棕色边框在不同背景色调下看起来一致。其次，高度阴影使用极大模糊值（28px、70px）配合适度透明度（0.14、0.1），创造漫射的大气提升而非硬边投影。卡片不是感觉浮在页面上方——而是感觉页面温柔地为它们开了一个空间。

### 装饰性深度
- 温暖奶油色表面变化创造微妙的色调深度而无需阴影
- 10% 和 20% 的 oklab 边框创造边缘定义的光谱
- 无生硬分割线——通过背景色调变化和间距实现区块分离

## 7. 交互与动效

### 悬停状态
- 按钮：文字颜色变为 `--color-error`（`#cf2d56`）——一种独特的温暖深红色，表示交互性
- 链接：颜色变为强调橙色（`#f54e00`）或下划线装饰配 `rgba(38, 37, 30, 0.4)`
- 卡片：悬停时阴影加深（环境 → 悬浮）

### 聚焦状态
- 基于阴影的聚焦：`rgba(0,0,0,0.1) 0px 4px 12px` 用于基于深度的聚焦指示
- 边框聚焦：`oklab(0.263 / 0.2)`（20% 边框）用于输入/表单聚焦
- 所有聚焦状态保持一致的温暖色调——无冷蓝色聚焦环

### 过渡
- 颜色过渡：150ms ease 用于文字/背景颜色变化
- 阴影过渡：200ms ease 用于高度变化
- 变换：微妙的缩放或平移用于交互反馈

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|-------|-------------|
| 移动端 | <600px | 单列，减少内边距，堆叠导航 |
| 小平板 | 600-768px | 2 列网格开始 |
| 平板 | 768-900px | 扩展卡片网格，侧边栏出现 |
| 小桌面 | 900-1279px | 完整布局形成 |
| 桌面 | >1279px | 完整布局，最大内容宽度 |

### 触摸目标
- 按钮使用舒适的内边距（6px-14px 垂直，8px-14px 水平）
- 药丸按钮保持适合点击的大小，3px-10px 内边距
- 导航链接 14px，间距足够便于触摸

### 折叠策略
- 英雄区：72px CursorGothic → 36px → 26px 在较小屏幕，保持比例字间距
- 导航：水平链接 → 移动端汉堡菜单
- 功能卡片：3 列 → 2 列 → 单列堆叠
