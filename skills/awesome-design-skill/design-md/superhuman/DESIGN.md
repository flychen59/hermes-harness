# 设计系统：Superhuman

## 1. 视觉主题与氛围

Superhuman 的网站宛如开启一封奢华信封——以白色为主色调，极其干净，仅用一抹戏剧性的色彩吸引目光。Hero 区域采用电影质感的紫色渐变，深邃的暮色洗染 `#1b1938`，唤起黎明前那一刻的感觉，覆盖着自信的白色文字。在这戏剧性的开场之下，网站其余部分几乎完全是白色画布配深炭灰色文字，营造出鲜明而精致的阅读体验。

字体是真正的标志性元素：Super Sans VF，一款自定义可变字体，采用非常规的字重档位（460、540、600、700），介于传统字重类别之间。460 字重——略重于 Regular 但轻于 Medium——是主力军，创造出的文字比典型的 400 字重更有自信，却从不咄咄逼人。紧凑的行高（展示文字为 0.96）将标题压缩成密集、有力的块状，而正文文字采用宽松的 1.50 行高，提供通透的可读性。这种压缩力量与呼吸空间之间的张力，定义了 Superhuman 的字体表达。

设计理念是通过最少的装饰展现最大的自信。使用温暖的奶油色按钮（`#e9e5dd`）而非明亮的 CTA，几乎没有边框和阴影，薰衣草紫色（`#cbb7fb`）作为唯一的强调色。这是一款以奢侈品品牌方式自我营销的生产力工具——每个像素都有其存在的理由，没有什么是纯粹的装饰。品牌命名惯例延伸到颜色：主色调紫色被称为 "Mysteria"，有意模糊蓝色与紫色的界限。

**关键特征：**
- 深紫色渐变 Hero（`#1b1938`）与主要白色内容区域形成对比
- Super Sans VF 可变字体采用非标准字重档位（460、540、600、700）——介于传统字重类别之间
- 超紧凑的展示行高（0.96）创造压缩、有力的标题
- 温暖的奶油色（`#e9e5dd`）按钮替代明亮/饱和的 CTA——低调奢华
- 薰衣草紫色（`#cbb7fb`）作为唯一的强调色——柔和、亲切的紫色
- 最小化的圆角规范：仅使用 8px 和 16px——无微圆角，无药丸形状
- 产品截图主导内容——UI 以最少的周边装饰自我展示

## 2. 色彩规范与角色

### 主色
- **Mysteria Purple**（`#1b1938`）：Hero 渐变背景，介于蓝紫之间的深紫色——品牌最深沉的表达
- **Lavender Glow**（`#cbb7fb`）：主要强调色和高亮色——柔和的紫色，用于强调、装饰元素和交互高亮
- **Charcoal Ink**（`#292827`）：浅色表面上的主要文字和标题颜色——带有淡淡棕色调的温暖近黑色

### 辅助色与强调色
- **Amethyst Link**（`#714cb6`）：带下划线的链接文字——中调紫色，与品牌调色板相连同时暗示交互性
- **Translucent White**（`color(srgb 1 1 1 / 0.95)`）：Hero 覆盖文字——95% 不透明度的近白色，用于深色表面的层次叠加
- **Misted White**（`color(srgb 1 1 1 / 0.8)`）：深色表面的次要文字——80% 不透明度的白色，用于 Hero 渐变上的层次区分

### 表面与背景
- **Pure White**（`#ffffff`）：主要页面背景——所有内容区域的主导画布颜色
- **Warm Cream**（`#e9e5dd`）：按钮背景——温暖的奶油色，避免纯灰色的冷调
- **Parchment Border**（`#dcd7d3`）：卡片和分割线边框——带有微粉色调的温暖浅灰色

### 中性色与文字
- **Charcoal Ink**（`#292827`）：白色表面上的主要标题和正文文字
- **Amethyst Link**（`#714cb6`）：带下划线装饰的内容链接
- **Translucent White 95%**（`color(srgb 1 1 1 / 0.95)`）：深色/紫色表面的主要文字
- **Translucent White 80%**（`color(srgb 1 1 1 / 0.8)`）：深色/紫色表面的次要文字

### 语义与强调
- Superhuman 在色彩运用上极度克制——Lavender Glow（`#cbb7fb`）是唯一真正的强调色
- 交互状态通过透明度变化和下划线装饰传达，而非颜色变化
- 温暖的奶油色按钮调色板避免任何饱和的语义颜色（营销页面上没有红色错误提示或绿色成功提示）

### 渐变系统
- **Hero Gradient**：从 `#1b1938` 开始的深紫色渐变，在 Hero 区域过渡穿越紫色到暮色的色调——整个网站最具戏剧性的视觉元素
- **Content Transition**：渐变消融入白色内容区域，当用户滚动时创造电影般的幕布升起效果
- 营销网站上没有其他渐变——Hero 渐变是单一而戏剧性的姿态

## 3. 字体规范

### 字体家族
- **展示与正文**：`Super Sans VF`——采用非标准字重轴的自定义可变字体。后备字体：`system-ui, -apple-system, Segoe UI, Roboto, Oxygen, Ubuntu, Cantarell, Fira Sans, Droid Sans, Helvetica Neue`
- **产品 UI**（品牌中提及）：来自 Luzi Type 的 `Messina Sans` / `Messina Serif` / `Messina Mono`——用于产品本身的无衬线到衬线过渡

### 原则
- **非标准字重轴**：字重 460 和 540 故意介于传统 Regular（400）和 Medium（500）之间，创造出一种微妙"偏离"但自信的字体质感——比预期略重，却从不显得粗体
- **极度展示压缩**：展示标题在 0.96 行高下几乎将行叠在一起，创造出建筑感十足的密集字体块
- **正文宽松度**：相比之下，正文文字在 1.50 行高下极其宽敞，确保在密集标题冲击后的舒适阅读
- **选择性负字间距**：字间距被精心应用——48px 标题为 -1.32px，28px 特性标题为 -0.63px，但正文文字为 0px。文字越大，字间距越紧
- **可变字体效率**：单个字体文件服务于所有字重变化（460-700），实现平滑的字重过渡和微调

## 4. 组件样式

### 按钮
- **Warm Cream 主要按钮**：`#e9e5dd` 背景，Charcoal Ink（`#292827`）文字，微妙的圆角（8px 半径），无可见边框。标志性 CTA——温暖、柔和、奢华而非咄咄逼人
- **深色主要按钮**（浅色区块）：`#292827` 背景配白色文字，8px 半径——温暖奶油色的反向版本，用于对比区块
- **幽灵 / 文字链接**：无背景，下划线装饰，Amethyst Link（`#714cb6`）或 Charcoal Ink 颜色，视上下文而定
- **Hero CTA**：深紫色渐变上的 Warm Cream——奶油色在 `#1b1938` 背景上戏剧性地凸显
- **悬停**：微妙的透明度或亮度变化——无戏剧性的颜色转变

### 卡片与容器
- **内容卡片**：白色背景，Parchment Border（`#dcd7d3`）1px 边框，16px 圆角——干净简约
- **深色表面卡片**：深色区块使用 `#292827` 边框，保持温暖中性调
- **Hero 表面**：紫色渐变上的半透明白色边框（`rgba(255, 255, 255, 0.2)`）——幽灵般的围合
- **产品截图卡片**：大型产品 UI 图片配干净边缘，最小框架——产品本身即是视觉
- **悬停**：最小状态变化——一致与平静胜过花哨的交互

### 输入与表单
- 营销网站上表单存在感最小——Superhuman 直接引导用户注册
- 深色边框输入框，使用 Charcoal Ink 边框和温暖色调的占位符文字
- 聚焦：边框强调增强，可能从 Parchment Border 变为 Charcoal Ink

### 导航
- **顶部导航**：内容区域为干净的白色背景，Hero 渐变上为透明
- **导航链接**：Super Sans VF 16px，字重 460/600 用于层级区分
- **CTA 按钮**：导航中的 Warm Cream（`#e9e5dd`）药丸形状——微妙，不抢眼
- **粘性行为**：导航在滚动时保持固定，伴随背景过渡
- **移动端**：折叠为汉堡菜单，简化布局

### 图片处理
- **产品截图**：大型、主导的产品 UI 图片展示邮件界面——产品即是主角
- **生活方式摄影**：Hero 区域单一戏剧性图片（紫色/红色渐变前的剪影）——电影感与编辑风格
- **全宽呈现**：截图跨越整个容器宽度，配微妙阴影或无边框
- **宽高比**：产品截图采用宽屏横版比例（约 16:9）
- **色彩整合**：截图经过精心调色，与紫色到白色的页面流程和谐统一

### 推荐 / 社交证明
- "Your Superhuman suite" 区域展示产品特性网格
- 特性描述配产品截图——通过演示而非引言来证明
- 干净的网格布局，卡片尺寸一致

## 5. 布局原则

### 间距系统
- **基础单位**：8px
- **规范**：2px、4px、6px、8px、12px、16px、18px、20px、24px、28px、32px、36px、40px、48px、56px
- **章节内边距**：主要章节间垂直间距 48px-80px
- **卡片内边距**：内部间距 16px-32px
- **组件间隙**：相关元素间 8px-16px

### 网格与容器
- **最大宽度**：约 1200px 内容容器，居中
- **列模式**：全宽 Hero，关键信息居中单列，特性卡片 2-3 列网格
- **特性网格**："Your Superhuman suite" 产品展示采用均匀列分布

### 留白理念
- **自信的空白**：章节间宽裕的留白传递高端定位——每个元素都有呼吸空间
- **产品即内容**：大型产品截图填充空间，其他网站会用营销文案填充
- **递进密度**：Hero 区域宽敞而富有电影感，内容区域随特性网格变密，然后为 CTA 再次开放

### 圆角规范
- **8px**：按钮、行内元素（`span`、`button`、`div`）——通用的较小半径
- **16px**：卡片、链接、较大容器（`a`、卡片元素）——通用的较大半径
- 整个系统只有两个圆角值——极简主义。无微圆角（2px），无药丸形状（50px+）

## 6. 深度与层级

| 层级 | 处理方式 | 用途 |
|------|----------|------|
| 层级 0（扁平） | 无阴影，白色背景 | 主要页面画布，大多数内容表面 |
| 层级 1（边框） | `1px solid #dcd7d3`（Parchment Border） | 卡片围合，章节分割线 |
| 层级 2（深色边框） | `1px solid #292827` | 头部元素，深色区块分隔符 |
| 层级 3（光晕） | 微妙阴影（检测到 6 个阴影定义） | 产品截图容器，提升的卡片 |
| 层级 4（Hero 深度） | `rgba(255, 255, 255, 0.2)` 透明边框 | 深紫色渐变 Hero 上的元素 |

### 阴影理念
Superhuman 在营销网站上的层级系统极其克制。深度主要通过以下方式传达：
- **边框围合**：温暖色调的边框（`#dcd7d3`）在 1px 下创造温和的分离
- **色彩对比**：Hero 渐变通过颜色变化而非阴影创造巨大的深度感
- **产品截图**：截图本身通过在扁平页面中展示分层 UI 来创造深度
- **透明度分层**：Hero 渐变上的半透明白色创造大气深度层次

### 装饰深度
- **Hero 渐变**：`#1b1938` → 白色的渐变过渡是主要的深度手段——电影般的幕布效果
- **薰衣草强调**：`#cbb7fb` Lavender Glow 元素漂浮在深色渐变之上，创造星空/大气效果
- **无毛玻璃效果**：尽管有半透明边框，但没有模糊/磨砂玻璃效果
- **摄影深度**：Hero 剪影图片创造自然的大气深度，无需人工 CSS

## 7. 设计守则

### 应该做的
- 使用 Super Sans VF 的 460 字重作为默认——它比 Regular 略重，这是品牌的字体标志性特征
- 将展示标题保持在 0.96 行高——这种压缩是有意为之且有力的
- 使用 Warm Cream（`#e9e5dd`）作为主要按钮——不是白色，不是灰色，特指温暖奶油色
- 将圆角限制在 8px（小）和 16px（大）——二元圆角系统是有意为之
- 仅在标题上应用负字间距（-0.63px 至 -1.32px）——正文文字保持 0px
- 使用 Lavender Glow（`#cbb7fb`）作为唯一的强调色——它是中性调色板之外唯一的色彩突破
- 让产品截图成为主要视觉内容——UI 自我展示
- 保持戏剧性的 Hero 渐变作为单一姿态——页面其余部分为白色

### 不应该做的
- 使用传统字重（400、500、600）——Superhuman 的 460 和 540 故意介于标准档位之间
- 添加明亮或饱和的 CTA 颜色（蓝色、绿色、红色）——按钮有意使用 Warm Cream 或 Charcoal 的柔和色调
- 引入 Lavender Glow 以外的额外强调色——调色板有意限制为一种强调色
- 大量使用阴影——深度来自边框、色彩对比和摄影，而非 box-shadows
- 在正文文字上使用紧凑行高——展示文字压缩（0.96）但正文宽松（1.50）
- 添加装饰元素、图标或插画——Superhuman 依赖产品 UI 和极简字体
- 创建药丸形状按钮——系统使用 8px 半径，而非圆形药丸
- 使用纯黑色（`#000000`）作为文字——Charcoal Ink（`#292827`）更温暖更柔和

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|------|----------|
| 移动端 | <768px | 单列，Hero 文字缩小至约 36px，堆叠特性卡片，汉堡导航 |
| 平板 | 768px-1024px | 2 列特性网格开始，Hero 文字约 48px，导航部分可见 |
| 桌面 | 1024px-1440px | 完整布局，64px Hero 展示，多列特性网格，完整导航 |
| 大桌面 | >1440px | 最大宽度容器居中，宽裕的侧边距 |

### 触摸目标
- 按钮：8px 半径配舒适的填充——符合触摸目标指南
- 导航链接：16px 文字配充足的周边填充
- 移动端 CTA：全宽 Warm Cream 按钮，便于拇指触达
- 链接：下划线装饰提供清晰的点击提示

### 折叠策略
- **导航**：完整水平导航 → 移动端汉堡菜单
- **Hero 文字**：64px 展示 → 48px → 约 36px 跨断点变化
- **特性网格**：多列产品展示 → 2 列 → 单列堆叠
- **产品截图**：在容器内缩放，保持横版比例
- **章节间距**：按比例缩小——宽裕的桌面边距在移动端压缩

### 图片行为
- 产品截图响应式缩放，保持宽高比
- Hero 剪影图片裁剪或缩放——保持戏剧性构图
- 无艺术指导变化——所有断点使用相同构图
- 首屏以下的产品截图可能采用懒加载
