# 设计系统：Apple

## 1. 视觉主题与氛围

Apple 的网站是控制戏剧的大师级作品——大片纯黑和近白作为电影般的背景，产品像画廊里的雕塑一样被拍摄。设计哲学本质上极简：每个像素都为产品服务，界面本身退居幕后直到变得不可见。这不是美学偏好上的极简主义；这是对对象的敬畏之情的极简主义。

排版是一切的锚点。San Francisco（大尺寸用 SF Pro Display，正文用 SF Pro Text）是 Apple 的专有字体，采用光学尺寸设计，可根据点大小自动调整字形。在展示尺寸（56px），字重 600 配紧凑行高 1.07 和微妙负字间距（-0.28px）创造感觉像机器制造而非排版的标题——精准、自信、直接得毫不客气。在正文尺寸（17px），字间距略微放宽（-0.374px），行高打开到 1.47，创造既舒适又从不松懈的阅读节奏。

色彩故事极其二元。产品章节在纯黑（`#000000`）背景配白文本和浅灰（`#f5f5f7`）背景配近黑文本（`#1d1d1f`）之间交替。这创造了电影般的节奏——深色章节感觉沉浸和高端，浅色章节感觉开放和信息性。唯一的色彩强调是 Apple Blue（`#0071e3`），专门保留给交互元素：链接、按钮和聚焦状态。这一单一强调色在中性色海洋中使每个可点击元素都有不可否认的可见性。

**关键特征：**
- SF Pro Display/Text 配光学尺寸——字形自动适应尺寸上下文
- 二元明/暗章节节奏：黑色（`#000000`）与浅灰（`#f5f5f7`）交替
- 单一强调色：Apple Blue（`#0071e3`）专门保留给交互元素
- 产品即英雄摄影在纯色场上——无渐变、无纹理、无干扰
- 极紧凑标题行高（1.07-1.14）创造压缩、广告牌般的冲击
- 全宽章节布局配居中内容——视口即画布
- 胶囊状 CTA（980px 圆角）创造柔和、亲切的行动按钮
- 章节间慷慨留白让每个产品时刻呼吸

## 2. 色彩调色板与角色

### 主色
- **Pure Black**（`#000000`）：英雄章节背景，沉浸式产品展示。最暗画布展示最亮产品。
- **Light Gray**（`#f5f5f7`）：交替章节背景，信息区域。不是白色——微妙的蓝灰色调防止无菌感。
- **Near Black**（`#1d1d1f`）：浅色背景上的主文本，深色按钮填充。比纯黑略暖，便于舒适阅读。

### 交互色
- **Apple Blue**（`#0071e3`）：`--sk-focus-color`，主 CTA 背景，聚焦环。界面中唯一的彩色。
- **Link Blue**（`#0066cc`）：`--sk-body-link-color`，行内文本链接。比 Apple Blue 略暗，便于文本级可读性。
- **Bright Blue**（`#2997ff`）：深色背景上的链接。更高亮度以在黑色章节上形成对比。

### 文本色
- **White**（`#ffffff`）：深色背景上的文本，蓝色/深色 CTA 上的按钮文本。
- **Near Black**（`#1d1d1f`）：浅色背景上的主要正文。
- **Black 80%**（`rgba(0, 0, 0, 0.8)`）：次要文本，浅色背景上的导航项。略微柔化。
- **Black 48%**（`rgba(0, 0, 0, 0.48)`）：三级文本，禁用状态，轮播控件。

### 表面与深色变体
- **Dark Surface 1**（`#272729`）：深色章节中的卡片背景。
- **Dark Surface 2**（`#262628`）：深色上下文中微妙的表面变化。
- **Dark Surface 3**（`#28282a`）：深色背景上的提升卡片。
- **Dark Surface 4**（`#2a2a2d`）：最高深色表面提升。
- **Dark Surface 5**（`#242426`）：最深深色表面色调。

### 按钮状态
- **Button Active**（`#ededf2`）：浅色按钮的激活/按下状态。
- **Button Default Light**（`#fafafc`）：搜索/筛选按钮背景。
- **Overlay**（`rgba(210, 210, 215, 0.64)`）：媒体控件遮罩，覆盖层。
- **White 32%**（`rgba(255, 255, 255, 0.32)`）：深色模态关闭按钮的悬停状态。

### 阴影
- **Card Shadow**（`rgba(0, 0, 0, 0.22) 3px 5px 30px 0px`）：产品卡片的柔和、漫射提升。偏移和宽模糊创造自然、摄影般的阴影。

## 3. 排版规则

### 字体家族
- **Display**：`SF Pro Display`，后备字体：`SF Pro Icons, Helvetica Neue, Helvetica, Arial, sans-serif`
- **正文**：`SF Pro Text`，后备字体：`SF Pro Icons, Helvetica Neue, Helvetica, Arial, sans-serif`
- SF Pro Display 用于 20px 及以上；SF Pro Text 针对 19px 及以下优化。

### 原则
- **光学尺寸即哲学**：SF Pro 自动在 Display 和 Text 光学尺寸之间切换。Display 版本具有更宽的字间距和更细的笔画，针对大尺寸优化；Text 版本更紧凑和坚固，针对小尺寸。这意味着字体实际上根据上下文改变其 DNA。
- **字重克制**：刻度跨越 300（轻）到 700（粗），但大多数文本位于 400（常规）和 600（半粗）。字重 300 仅出现在大型装饰性文本上。字重 700 罕见，仅用于粗体卡片标题。
- **所有尺寸负字间距**：与大多数仅追踪标题的系统不同，Apple 甚至在正文尺寸也应用微妙负字间距（17px 时 -0.374px，14px 时 -0.224px，12px 时 -0.12px）。这创造了普遍紧凑、高效的文本。
- **极端行高范围**：标题压缩到 1.07，而正文打开到 1.47，某些按钮上下文延伸到 2.41。这种戏剧性范围仅通过节奏创造清晰的视觉层级。

## 4. 组件样式

### 按钮

**Primary Blue (CTA)**
- 背景：`#0071e3`（Apple Blue）
- 文本：`#ffffff`
- 内边距：8px 15px
- 圆角：8px
- 边框：1px solid transparent
- 字体：SF Pro Text，17px，字重 400
- 悬停：背景略微变亮
- 激活：`#ededf2` 背景转换
- 聚焦：`2px solid var(--sk-focus-color, #0071E3)` 轮廓
- 用途：主要行动号召（"购买"、"购买 iPhone"）

**Primary Dark**
- 背景：`#1d1d1f`
- 文本：`#ffffff`
- 内边距：8px 15px
- 圆角：8px
- 字体：SF Pro Text，17px，字重 400
- 用途：次要 CTA，深色变体

**Pill Link (Learn More / Shop)**
- 背景：透明
- 文本：`#0066cc`（浅色背景）或 `#2997ff`（深色背景）
- 圆角：980px（完整胶囊）
- 边框：1px solid `#0066cc`
- 字体：SF Pro Text，14px-17px
- 悬停：下划线装饰
- 用途："了解更多"和"购买"链接——标志性的 Apple 行内 CTA

**Filter / Search Button**
- 背景：`#fafafc`
- 文本：`rgba(0, 0, 0, 0.8)`
- 内边距：0px 14px
- 圆角：11px
- 边框：3px solid `rgba(0, 0, 0, 0.04)`
- 聚焦：`2px solid var(--sk-focus-color, #0071E3)` 轮廓
- 用途：搜索栏，筛选控件

**Media Control**
- 背景：`rgba(210, 210, 215, 0.64)`
- 文本：`rgba(0, 0, 0, 0.48)`
- 圆角：50%（圆形）
- 激活：scale(0.9)，背景转换
- 聚焦：`2px solid var(--sk-focus-color, #0071e3)` 轮廓，白色背景，黑色文本
- 用途：播放/暂停，轮播箭头

### 卡片与容器
- 背景：`#f5f5f7`（浅色）或 `#272729`-`#2a2a2d`（深色）
- 边框：无（Apple 系统中边框罕见）
- 圆角：5px-8px
- 阴影：`rgba(0, 0, 0, 0.22) 3px 5px 30px 0px` 用于提升产品卡片
- 内容：居中，内边距慷慨
- 悬停：无标准悬停状态——卡片是静态的，其中的链接是交互的

### 导航
- 背景：`rgba(0, 0, 0, 0.8)`（半透明深色）配 `backdrop-filter: saturate(180%) blur(20px)`
- 高度：48px（紧凑）
- 文本：`#ffffff`，12px，字重 400
- 激活：悬停时下划线
- Logo：Apple 标志（SVG）居中或左对齐，17x48px 视口
- 移动端：折叠为汉堡菜单配全屏覆盖菜单
- 导航浮动在内容上方，无论章节背景如何都保持深色半透明玻璃效果

### 图片处理
- 产品在纯色场上（黑色或白色）——无背景、无上下文、只有对象
- 跨越整个视口宽度的全出血章节图片
- 产品摄影分辨率极高，阴影微妙
- 生活方式图片限制在圆角容器内（12px+ 圆角）

### 独特组件

**Product Hero Module**
- 全视口宽度章节配纯色背景（黑色或 `#f5f5f7`）
- 产品名称作为主标题（SF Pro Display，56px，字重 600）
- 下方一行描述，字重较轻
- 两个胶囊 CTA 并排："了解更多"（轮廓）和"购买" / "购买"（填充）

**Product Grid Tile**
- 对比背景上的方形或近方形卡片
- 产品图片占据瓷砖 60-70%
- 下方产品名称 + 一行描述
- 底部"了解更多"和"购买"链接对

**Feature Comparison Strip**
- 产品变体水平滚动
- 每个变体作为垂直卡片，包含图片、名称和关键规格
- 最小修饰——产品自己说话

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 刻度：2px, 4px, 5px, 6px, 7px, 8px, 9px, 10px, 11px, 14px, 15px, 17px, 20px, 24px
- 显著特征：刻度在小尺寸（2-11px）密集，有 1px 增量，然后以较大步长跳跃。这允许对排版和图标对齐进行精确微调。

### 网格与容器
- 最大内容宽度：约 980px（胶囊按钮中的"980px 圆角"呼应此宽度）
- 英雄：全视口宽度章节配居中内容块
- 产品网格：居中容器内 2-3 列布局
- 英雄时刻单列——一个产品、一个信息、全神贯注
- 无可见网格线或槽——间距创造隐含结构

### 留白哲学
- **电影般的呼吸空间**：每个产品章节占据完整视口高度（或接近）。产品之间的留白不是空的——它是电影中场景之间的停顿。
- **通过色块实现垂直节奏**：Apple 不单独使用间距来分隔章节，而是使用交替背景色（黑色、`#f5f5f7`、白色）。每次颜色变化标志新的"场景"。
- **内紧外松**：文本块紧凑设置（负字间距、紧行高），而周围空间广阔。这在密度和开放之间创造张力。

### Border Radius 刻度
- 微型（5px）：小容器，链接标签
- 标准（8px）：按钮，产品卡片，图片容器
- 舒适（11px）：搜索输入，筛选按钮
- 大型（12px）：特性面板，生活方式图片容器
- 完整胶囊（980px）：CTA 链接（"了解更多"、"购买"），导航胶囊
- 圆形（50%）：媒体控件（播放/暂停、箭头）

## 6. 深度与提升

| 级别 | 处理方式 | 用途 |
|-------|-----------|-----|
| Flat (Level 0) | 无阴影，纯色背景 | 标准内容章节，文本块 |
| Navigation Glass | `rgba(0,0,0,0.8)` 上的 `backdrop-filter: saturate(180%) blur(20px)` | 粘性导航栏——玻璃效果 |
| Subtle Lift (Level 1) | `rgba(0, 0, 0, 0.22) 3px 5px 30px 0px` | 产品卡片，浮动元素 |
| Media Control | `rgba(210, 210, 215, 0.64)` 背景配 scale 变换 | 播放/暂停按钮，轮播控件 |
| Focus (无障碍) | `2px solid #0071e3` 轮廓 | 所有交互元素的键盘聚焦 |

**阴影哲学**：Apple 极少使用阴影。主要阴影（`3px 5px 30px`，0.22 不透明度）柔和、宽阔、偏移——模仿漫射工作室灯光在物理对象下方投射自然阴影。这强化了"产品即物理雕塑"的隐喻。大多数元素完全没有阴影；提升来自背景色对比（深色背景上的深色卡片，或略不同灰色上的浅色卡片）。

### 装饰性深度
- 导航玻璃：半透明、模糊的导航栏是最可识别的深度元素，创造滚动内容上方浮动 UI 的感觉
- 章节颜色过渡：深度通过黑色和浅灰章节交替暗示，而非阴影
- 产品摄影阴影：产品本身在摄影中投射阴影，所以 UI 不需要添加合成的

## 7. 宜与忌

### 宜
- 20px+ 使用 SF Pro Display，20px 以下使用 SF Pro Text——尊重光学尺寸边界
- 在所有文本尺寸应用负字间距（不仅是标题）——Apple 全面紧凑追踪
- 仅将 Apple Blue（`#0071e3`）用于交互元素——它必须是唯一强调色
- 黑色和浅灰（`#f5f5f7`）章节背景交替以创造电影节奏
- CTA 链接使用 980px 胶囊圆角——标志性的 Apple 链接形状
- 产品图片保持在纯色场上，无竞争性视觉元素
- 粘性导航使用半透明深色玻璃（`rgba(0,0,0,0.8)` + blur）
- 将标题行高压缩到 1.07-1.14——Apple 标题以紧凑著称

### 忌
- 不要引入额外强调色——整个色彩预算都花在蓝色上
- 不要使用重阴影或多层阴影——Apple 的阴影系统是一个柔和漫射阴影或没有
- 不要在卡片或容器上使用边框——Apple 几乎从不使用可见边框（特定按钮除外）
- 不要对 SF Pro 应用宽字间距——它设计为在每个尺寸紧凑运行
- 不要使用字重 800 或 900——最大是 700（粗体），即使那样也很罕见
- 不要在背景添加纹理、图案或渐变——仅纯色
- 不要让导航不透明——玻璃模糊效果对 Apple UI 身份至关重要
- 不要居中对齐正文——Apple 正文左对齐；仅标题居中
- 不要在矩形元素上使用大于 12px 的圆角（980px 仅用于胶囊）

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|-------|-------------|
| Small Mobile | <360px | 支持的最小尺寸，单列 |
| Mobile | 360-480px | 标准移动布局 |
| Mobile Large | 480-640px | 更宽单列，更大图片 |
| Tablet Small | 640-834px | 2 列产品网格开始 |
| Tablet | 834-1024px | 完整平板布局，扩展导航 |
| Desktop Small | 1024-1070px | 标准桌面布局开始 |
| Desktop | 1070-1440px | 完整布局，最大内容宽度 |
| Large Desktop | >1440px | 居中配慷慨边距 |

### 触控目标
- 主 CTA：8px 15px 内边距创造约 44px 触摸高度
- 导航链接：48px 高度，间距充足
- 媒体控件：50% 圆角圆形按钮，最小 44x44px
- "了解更多"胶囊：内边距慷慨，便于舒适点击

### 折叠策略
- 英雄标题：56px Display → 40px → 移动端 28px，按比例保持紧凑行高
- 产品网格：3 列 → 2 列 → 单列堆叠
- 导航：完整水平导航 → 紧凑移动菜单（汉堡）
- 产品英雄模块：所有尺寸保持全出血，文本按比例缩小
- 章节背景：所有断点保持全宽色块——电影节奏永不中断
- 图片尺寸：产品按比例缩放，永不裁剪——产品轮廓神圣不可侵犯

### 图片行为
- 产品摄影在所有断点保持纵横比
- 英雄产品图片缩小但保持居中
- 全出血章节背景在每个尺寸持续
- 生活方式图片可能在移动端裁剪但保持圆角
- 折叠下方产品图片懒加载

### 迭代指南
1. 每个交互元素获得 Apple Blue（`#0071e3`）——无其他强调色
2. 章节背景交替：黑色用于沉浸时刻，`#f5f5f7` 用于信息时刻
3. 排版光学尺寸：20px+ 使用 SF Pro Display，以下使用 SF Pro Text——永不混合
4. 所有尺寸负字间距：56px 时 -0.28px，17px 时 -0.374px，14px 时 -0.224px，12px 时 -0.12px
5. 导航玻璃效果（半透明深色 + blur）不可协商——它定义了 Apple 网页体验
6. 产品始终出现在纯色场上——永不出现在英雄模块的渐变、纹理或生活方式背景上
7. 阴影罕见且始终柔和：`3px 5px 30px 0.22 opacity` 或根本没有
8. 胶囊 CTA 使用 980px 圆角——这创造了标志性的 Apple 圆角矩形看起来像胶囊的形状
