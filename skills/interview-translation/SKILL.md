# 采访文档中译润色

将采访文档（意大利语、英语或其他语言）翻译为自然、真诚的中文。

## 翻译原则

### 语气定位
- **放松真诚的聊天氛围**，像两个朋友在咖啡馆聊天，不是学术答辩
- 受访者在有感而发地讲自己的人生经历，不是在背稿子
- 可以用口语词："就是""其实""说实话""确实""挺""反正""我觉得吧"

### 禁止表达（抽象/书面/学术）
以下词语**禁止使用**，必须替换为具体说法：

| 禁止词 | 替换方向 |
|--------|----------|
| 标签 | 根据上下文说具体是什么（"固定的看法""总觉得我就是那个中国来的移民"） |
| 简化 | "不把人当个体看""一概而论""把复杂的事说得很片面" |
| 归类 | "直接把我当成XX" |
| 机制 | 说具体是什么过程 |
| 维度 | 说具体是哪个方面 |
| 符号 | 说具体是什么 |
| 框架 | "这套思路""这个角度" |
| 结构性 | "系统性的""整个体系的问题" |
| 连续体 | "一个完整的我，只是不同场合不同部分更突出" |
| 协商身份 | "在身份上反复纠结""不断调整自己" |
| 认知 | "认识""想法""看法" |
| 迁徙 | 用"移民"（迁徙形容动物，不能用来形容人） |
| 种族化 | 直接说"种族歧视"（种族化就是种族歧视） |
| 深刻影响 | "影响特别大""对我冲击很大" |
| 他者 | "外人" |
| 表征/再现 | "呈现""描述" |
| 系统的（形容支持/安排时） | "帮我过渡一下的安排" |
| 预设身份 | "给我定了个人设" |
| 零碎/碎片化 | "零零碎碎的" |
| 代表性缺失 | "一直缺了我们的声音" |
| 产出知识 | "讲自己的事""去理解世界" |

### 生动具体原则
- **每句话都要具体**，不能模糊含糊
- "固定的标签" → 说具体是什么标签
- "被贴标签" → "被人用固定的看法套住"
- "被简化" → "被一概而论""不把人当个体看"
- "只看肤色长相就定义了一个人"
- "别人总觉得我就应该温顺、安静、听话那种"

### 好的翻译范例
- ✅ "说实话这个变化真的挺大的"
- ✅ "这真的让我开始想，我到底是谁"
- ✅ "头几个月我说话都费劲"
- ✅ "别人总觉得我就应该温顺、安静、听话那种"
- ✅ "把我当成了一个活教材"
- ✅ "老爱替我们说话、而不是真正听我们说的"

### 差的翻译范例（要避免）
- ❌ "这个转变非常剧烈，尤其是在语言和社会层面"
- ❌ "这深刻影响了我的身份认知"
- ❌ "协商我的身份"
- ❌ "强化了我作为'他者'的感觉"
- ❌ "种族化通过持续的目光注视来体现"

## 输出格式

```
章节标题（如"第一部分——个人经历与移民"）

提问人[采访人姓名]: [翻译内容]
回答R: [翻译内容]
```

- 章节标题单独一行，不带任何符号（🔹👤等全部删除）
- **提问不带序号**，格式为：提问人[采访人姓名]: [内容]（不是"提问人XX: 1. 你好"，而是"提问人XX: 你好"）
- 提问人和回答之间空一行
- 不插入中间分隔标题（如"受访者3——第四部分"）

## 工作流程

1. 读取源文件（支持 .docx / .txt）
2. 逐句翻译，确保语意明确不含糊
3. 每句翻译后检查：这句话像不像真人在说话？有没有抽象词？
4. 全部完成后做一遍通读检查
5. 输出到指定文件
