# 设计系统：Pinterest

## 1. 视觉主题与氛围

Pinterest 的网站是一个温暖、以灵感为导向的画布，将视觉发现视为一本生活方式杂志。设计采用柔和、微暖的白色背景，以 Pinterest Red（`#e60023`）作为单一、大胆的品牌强调色。与大多数科技平台的冷色调蓝色不同，Pinterest 的中性色调具有明显的暖色底色——灰色偏向橄榄色/沙色（`#91918c`、`#62625b`、`#e5e5e0`），而非冷峻的钢灰色，营造出一种温馨、手工艺般的氛围，邀请用户浏览。

排版使用 Pin Sans——一种定制的专有字体，包含广泛的 fallback 字体栈，包括日文字体，体现了 Pinterest 的全球化覆盖。在展示级别（70px，weight 600），Pin Sans 创造出大型、引人注目的标题。在较小尺寸下，系统紧凑：按钮为 12px，说明文字为 12–14px。CSS 变量命名系统（`--comp-*`、`--sema-*`、`--base-*`）揭示了复杂的三层设计令牌架构：组件级、语义级和基础级令牌。

Pinterest 的独特之处在于其宽大的 border-radius 系统（12px–40px，加上 50% 用于圆形）和暖色调按钮背景。次要按钮（`#e5e5e0`）具有明显的温暖、沙质般的色调，而非冷灰色。主要红色按钮使用 16px 圆角——圆润但不是胶囊形。结合温暖的徽章背景（`hsla(60,20%,98%,.5)`——一种微妙的黄色暖色涂层）和以摄影为主的布局，结果是设计感觉手工艺化和个性化，而非企业化和冷漠。

**关键特征：**
- 暖白色画布配橄榄色/沙色调中性色——温馨而非冷峻
- Pinterest Red（`#e60023`）作为单一大胆强调色——从不微妙，总是自信
- Pin Sans 定制字体配全球化 fallback 字体栈（包括 CJK）
- 三层令牌架构：`--comp-*` / `--sema-*` / `--base-*`
- 温暖的次要表面：沙灰色（`#e5e5e0`），暖色徽章（`hsla(60,20%,98%,.5)`）
- 宽大的 border-radius：16px 标准，大型容器最高 40px
- 摄影优先的内容——Pins/图片是主要视觉元素
- 深近紫色文本（`#211922`）——温暖，带有一丝紫红色调

## 2. 色彩系统与角色

### 主要品牌色
- **Pinterest Red**（`#e60023`）：主要 CTA，品牌强调色——大胆、自信的红色
- **Green 700**（`#103c25`）：`--base-color-green-700`，成功/自然强调色
- **Green 700 Hover**（`#0b2819`）：`--base-color-hover-green-700`，按下状态的绿色

### 文本色
- **Plum Black**（`#211922`）：主要文本——带有紫红色调的暖近黑色
- **Black**（`#000000`）：次要文本，按钮文本
- **Olive Gray**（`#62625b`）：次要描述，弱化文本
- **Warm Silver**（`#91918c`）：`--comp-button-color-text-transparent-disabled`，禁用文本，输入框边框
- **White**（`#ffffff`）：深色/彩色表面上的文本

### 交互色
- **Focus Blue**（`#435ee5`）：`--comp-button-color-border-focus-outer-transparent`，焦点环
- **Performance Purple**（`#6845ab`）：`--sema-color-hover-icon-performance-plus`，性能功能
- **Recommendation Purple**（`#7e238b`）：`--sema-color-hover-text-recommendation`，AI 推荐
- **Link Blue**（`#2b48d4`）：链接文本颜色
- **Facebook Blue**（`#0866ff`）：`--facebook-background-color`，社交登录
- **Pressed Blue**（`#617bff`）：`--base-color-pressed-blue-200`，按下状态

### 表面与边框
- **Sand Gray**（`#e5e5e0`）：次要按钮背景——温暖、手工艺感
- **Warm Light**（`#e0e0d9`）：圆形按钮背景，徽章
- **Warm Wash**（`hsla(60, 20%, 98%, 0.5)`）：`--comp-badge-color-background-wash-light`，微妙暖色徽章背景
- **Fog**（`#f6f6f3`）：浅色表面（50% 透明度）
- **Border Disabled**（`#c8c8c1`）：`--sema-color-border-disabled`，禁用边框
- **Hover Gray**（`#bcbcb3`）：`--base-color-hover-grayscale-150`，悬停边框
- **Dark Surface**（`#33332e`）：深色区域背景

### 语义色
- **Error Red**（`#9e0a0a`）：复选框/表单错误状态

## 3. 排版规则

### 字体族
- **主要字体**：`Pin Sans`，fallback：`-apple-system, system-ui, Segoe UI, Roboto, Oxygen-Sans, Apple Color Emoji, Segoe UI Emoji, Segoe UI Symbol, Ubuntu, Cantarell, Fira Sans, Droid Sans, Helvetica Neue, Helvetica, ヒラギノ角ゴ Pro W3, メイリオ, Meiryo, ＭＳ Ｐゴシック, Arial`

### 层级结构

| 角色 | 字体 | 大小 | 字重 | 行高 | 字间距 | 备注 |
|------|------|------|------|------|--------|------|
| 展示英雄标题 | Pin Sans | 70px (4.38rem) | 600 | normal | normal | 最大冲击力 |
| 区域标题 | Pin Sans | 28px (1.75rem) | 700 | normal | -1.2px | 负字间距 |
| 正文 | Pin Sans | 16px (1.00rem) | 400 | 1.40 | normal | 标准阅读 |
| 说明文字粗体 | Pin Sans | 14px (0.88rem) | 700 | normal | normal | 强调元数据 |
| 说明文字 | Pin Sans | 12px (0.75rem) | 400–500 | 1.50 | normal | 小文本，标签 |
| 按钮 | Pin Sans | 12px (0.75rem) | 400 | normal | normal | 按钮标签 |

### 原则
- **紧凑的字型比例**：范围从 12px 到 70px，跨度巨大——大多数功能文本为 12–16px，创造密集的、应用式的信息层级。
- **温暖的字重分布**：标题使用 600–700，正文使用 400–500。没有超轻字重——字体总是感觉充实。
- **标题使用负字间距**：28px 标题使用 -1.2px 字间距，创造紧凑、亲切的区域标题。
- **单一字体族**：Pin Sans 处理所有内容——未检测到次要展示或等宽字体。

## 4. 组件样式

### 按钮

**主要红色按钮**
- 背景：`#e60023`（Pinterest Red）
- 文本：`#000000`（黑色——在红色上使用黑色是独特的选择）
- 内边距：6px 14px
- 圆角：16px（宽圆润，非胶囊形）
- 边框：`2px solid rgba(255, 255, 255, 0)`（透明）
- 焦点：通过 CSS 变量的语义边框 + outline

**次要沙色按钮**
- 背景：`#e5e5e0`（温暖沙灰色）
- 文本：`#000000`
- 内边距：6px 14px
- 圆角：16px
- 焦点：相同的语义边框系统

**圆形操作按钮**
- 背景：`#e0e0d9`（暖光色）
- 文本：`#211922`（plum black）
- 圆角：50%（圆形）
- 用途：Pin 操作，导航控件

**幽灵/透明按钮**
- 背景：透明
- 文本：`#000000`
- 无边框
- 用途：第三级操作

### 卡片与容器
- 摄影优先的 Pin 卡片，配宽大圆角（12px–20px）
- 大多数卡片没有传统 box-shadow
- 白色或暖雾色背景
- 部分图片容器有 8px 白色粗边框

### 输入框
- 邮箱输入：白色背景，`1px solid #91918c` 边框，16px 圆角，11px 15px 内边距
- 焦点：通过 CSS 变量的语义边框 + outline 系统

### 导航
- 白色或暖色背景上的简洁头部
- Pinterest logo + 居中搜索栏
- 导航链接使用 Pin Sans 16px
- 活跃状态使用 Pinterest Red 强调

### 图片处理
- Pin 风格的瀑布流网格（Pinterest 标志性布局）
- 圆角：图片上使用 12px–20px
- 摄影作为主要内容——每个 Pin 都是一张图片
- 精选图片容器上的粗白色边框（8px）

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 比例：4px, 6px, 7px, 8px, 10px, 11px, 12px, 16px, 18px, 20px, 22px, 24px, 32px, 80px, 100px
- 大跨度：32px → 80px → 100px 用于区域间距

### 网格与容器
- Pin 内容使用瀑布流网格（标志性布局）
- 居中内容区域，配宽大的最大宽度
- 全宽深色页脚
- 搜索栏作为主要导航元素

### 留白理念
- **灵感密度**：瀑布流网格紧密排列 Pins——内容密度本身就是价值主张。留白存在于区域之间，而非网格内部。
- **上方留白，下方密集**：Hero/特色区域获得宽大内边距；Pin 网格紧凑且沉浸式。

### Border Radius 比例
- 标准（12px）：小卡片，链接
- 按钮（16px）：按钮，输入框，中型卡片
- 舒适（20px）：特色卡片
- 大型（28px）：大型容器
- 区域（32px）：Tab 元素，大型面板
- 英雄区（40px）：Hero 容器，大型特色块
- 圆形（50%）：操作按钮，Tab 指示器

## 6. 深度与层级

| 层级 | 处理方式 | 用途 |
|------|----------|------|
| 平面（Level 0） | 无阴影 | 默认——Pins 依赖内容，而非阴影 |
| 微妙（Level 1） | 最小阴影（来自令牌） | 浮动覆盖层，下拉菜单 |
| 焦点（无障碍） | `--sema-color-border-focus-outer-default` 环 | 焦点状态 |

**阴影理念**：Pinterest 使用极少的阴影。瀑布流网格依赖内容（摄影）来创造视觉趣味，而非层级效果。深度来自表面颜色的温暖度和容器的宽大圆角。

## 7. 设计准则

### 应该做的
- 使用温暖的中性色（`#e5e5e0`、`#e0e0d9`、`#91918c`）——温暖的橄榄色/沙色调是身份特征
- 仅在主要 CTA 上应用 Pinterest Red（`#e60023`）——它大胆且单一
- 专门使用 Pin Sans——一种字体处理所有内容
- 应用宽大的 border-radius：按钮/输入框使用 16px，卡片使用 20px+
- 保持瀑布流网格密集——内容密度是价值所在
- 使用温暖的徽章背景（`hsla(60,20%,98%,.5)`）营造微妙的暖色涂层
- 使用 `#211922`（plum black）作为主要文本——它比纯黑色更温暖

### 不应该做的
- 不要使用冷灰色中性色——始终使用温暖/橄榄色调
- 不要使用纯黑色（`#000000`）作为主要文本——使用 plum black（`#211922`）
- 不要使用胶囊形按钮——16px 圆角是圆润但非胶囊形
- 不要添加重阴影——Pinterest 设计上是扁平的，深度来自内容
- 不要在卡片上使用小 border-radius（<12px）——宽大圆角是核心
- 不要引入额外的品牌颜色——红色 + 温暖中性色是完整的调色板
- 不要使用细字重——Pin Sans 最小使用 400

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|------|----------|
| 移动端 | <576px | 单列，紧凑布局 |
| 移动端大屏 | 576–768px | 2 列 Pin 网格 |
| 平板 | 768–890px | 扩展网格 |
| 桌面端小屏 | 890–1312px | 标准瀑布流网格 |
| 桌面端 | 1312–1440px | 完整布局 |
| 大型桌面 | 1440–1680px | 扩展网格列 |
| 超宽屏 | >1680px | 最大网格密度 |

### 折叠策略
- Pin 网格：5+ 列 → 3 → 2 → 1
- 导航：搜索栏 + 图标 → 简化的移动端导航
- 特色区域：并排 → 堆叠
- Hero：70px → 按比例缩小
- 页脚：深色多列 → 堆叠
