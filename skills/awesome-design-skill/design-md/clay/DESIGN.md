# 设计系统：Clay

## 1. 视觉主题与氛围

Clay 的网站是一场温暖、活泼的色彩盛宴，将 B2B 数据丰富化工作视为一种工艺，而非企业任务。设计语言建立在温暖的奶油色背景（`#faf9f7`）和燕麦色调边框（`#dad4c8`、`#eee9df`）之上，赋予每个表面手工纸张般的触感质地。在这个手工艺风格的画布上，鲜艳的色板调色板迸发出个性——Matcha 绿、Slushie 青、Lemon 金、Ube 紫、Pomegranate 粉、Blueberry 藏青和 Dragonfruit 洋红——每一个名字都像果汁吧的口味，而不是企业 UI 套件中的颜色名称。

字体排版以 Roobert 为核心，这是一款富有个性的几何无衬线字体，加载了丰富的 OpenType 样式集（`"ss01"`、`"ss03"`、`"ss10"`、`"ss11"`、`"ss12"`），赋予文字独特、略带古怪的个性。在展示尺寸（80px，字重 600）下，Roobert 使用激进的负字间距（-3.2px），将标题压缩成有力、广告牌式的声明。Space Mono 作为等宽字体伴侣，用于代码和技术标签，完成了工艺与科技的二元统一。

让 Clay 真正独特的是其悬停微动画：按钮悬停时轻微旋转（`rotateZ(-8deg)`），向上平移（`translateY(-80%)`），背景变为对比色板颜色，并投射硬边偏移阴影（`rgb(0,0,0) -7px 7px`）。这种有趣的悬停行为——按钮在交互时真正倾斜并跳跃——在 B2B 软件中创造了罕见的物理愉悦感。结合大圆角容器（24px–40px 圆角）、虚线边框与实线边框混用，以及包含内嵌高光的多层阴影系统，Clay 感觉像是一个由真正享受创造的人们打造的设计系统。

**核心特征：**
- 温暖奶油色画布（`#faf9f7`）配燕麦色调边框（`#dad4c8`）——手工艺风格，非临床感
- 命名色板调色板：Matcha、Slushie、Lemon、Ube、Pomegranate、Blueberry、Dragonfruit
- Roobert 字体配 5 组 OpenType 样式集——独特的几何个性
- 有趣的悬停动画：rotateZ(-8deg) + translateY(-80%) + 硬边偏移阴影
- Space Mono 用于代码和技术标签
- 大圆角：24px 卡片，40px 区块，1584px 药丸形
- 混合边框样式：同一界面中实线 + 虚线并存
- 带内嵌高光的多层阴影：`0px 1px 1px` + `-1px inset` + `-0.5px`

## 3. 字体排版规则

### 字体族
- **主字体**：`Roobert`，后备字体：`Arial`
- **等宽字体**：`Space Mono`
- **OpenType 特性**：所有 Roobert 文本启用 `"ss01"`、`"ss03"`、`"ss10"`、`"ss11"`、`"ss12"`（展示文本使用全部 5 组；正文/UI 使用 `"ss03"`、`"ss10"`、`"ss11"`、`"ss12"`）

### 原则
- **五组样式集作为身份标识**：Roobert 上 `"ss01"`、`"ss03"`、`"ss10"`、`"ss11"`、`"ss12"` 的组合创造了独特的字体个性。`ss01` 专用于标题和强调——正文文本省略它，通过字形变化创造微妙的层级。
- **激进的展示压缩**：80px 时 -3.2px，60px 时 -2.4px——最紧凑的展示字间距配以最宽松的正文行高（1.60 行高），创造戏剧性对比。
- **标题用字重 600，UI 用 500，正文用 400**：清晰的三级系统，每个字重都有严格的角色。
- **大写标签配正字间距**：12px 大写字母配 1.08px 字间距，创造系统化的导视模式。

## 4. 组件样式

### 按钮

**主按钮（透明配悬停动画）**
- 背景：透明（`rgba(239, 241, 243, 0)`）
- 文本：`#000000`
- 内边距：6.4px 12.8px
- 边框：无（或描边变体使用 `1px solid #717989`）
- 悬停：背景变为色板颜色（如 `#434346`），文本变白，`rotateZ(-8deg)`，`translateY(-80%)`，硬边阴影 `rgb(0,0,0) -7px 7px`
- 焦点：`rgb(20, 110, 245) solid 2px` 轮廓

**白色实心按钮**
- 背景：`#ffffff`
- 文本：`#000000`
- 内边距：6.4px
- 悬停：oat-200 色板颜色，动画旋转 + 阴影
- 用途：彩色区块上的主 CTA

**幽灵描边按钮**
- 背景：透明
- 文本：`#000000`
- 内边距：8px
- 边框：`1px solid #717989`
- 圆角：4px
- 悬停：dragonfruit 色板颜色，白色文本，动画旋转

### 卡片与容器
- 背景：奶油色画布上的 `#ffffff`
- 边框：`1px solid #dad4c8`（温暖燕麦色）或 `1px dashed #dad4c8`
- 圆角：12px（标准卡片），24px（功能卡片/图片），40px（区块容器/页脚）
- 阴影：`rgba(0,0,0,0.1) 0px 1px 1px, rgba(0,0,0,0.04) 0px -1px 1px inset, rgba(0,0,0,0.05) 0px -0.5px 1px`
- 使用色板调色板的彩色区块背景（matcha、slushie、ube、lemon）

### 输入框与表单
- 文本：`#000000`
- 边框：`1px solid #717989`
- 圆角：4px
- 焦点：`rgb(20, 110, 245) solid 2px` 轮廓

### 导航
- 奶油色背景上的粘性顶部导航
- 导航链接使用 Roobert 15px 字重 500
- Clay logo 左对齐
- CTA 按钮右对齐，药丸形圆角
- 底部边框：`1px solid #dad4c8`
- 移动端：767px 处折叠为汉堡菜单

### 图片处理
- 产品截图置于白卡配燕麦色边框
- 彩色插图区块配色板背景色
- 图片圆角 8px–24px
- 全宽彩色区块背景

### 独特组件

**色板颜色区块**
- 色板颜色背景的全宽区块（matcha 绿、slushie 青、ube 紫、lemon 金）
- 深色色板上用白字，浅色色板上用黑字
- 每个区块通过其颜色讲述独特的产品故事

**有趣的悬停按钮**
- 悬停时旋转 -8 度 + 向上平移
- 硬边偏移阴影（`-7px 7px`）而非柔和模糊
- 背景过渡为对比色板颜色
- 创造物理的、玩具般的交互质感

**虚线边框元素**
- 虚线边框（`1px dashed #dad4c8`）与实线边框并存
- 用于次要容器和装饰元素
- 增添手绘的、工艺般的质感

## 5. 布局原则

### 间距系统
- 基础单位：8px
- 比例：1px、2px、4px、6.4px、8px、12px、12.8px、16px、18px、20px、24px

### 网格与容器
- 内容最大宽度居中
- 功能区块在白卡和彩色色板背景之间交替
- 卡片网格：桌面端 2–3 列
- 全宽彩色区块打破网格
- 页脚使用大圆角 40px 容器

### 留白哲学
- **温暖、大方的呼吸感**：奶油色背景在内容块之间提供温暖的休憩。间距大方但不冷峻——感觉像精心布置的餐桌一样诱人。
- **色彩作为空间节奏**：交替的色板颜色区块通过色调而非仅靠留白创造视觉节奏。每个颜色区块都是自己的"房间"。
- **卡片内工艺般的密度**：在卡片内部，内容紧凑有序，与外部的大方间距形成对比。

### 圆角比例
- 尖锐（4px）：幽灵按钮、输入框
- 标准（8px）：小卡片、图片、链接
- 徽章（11px）：标签徽章
- 卡片（12px）：标准卡片、按钮
- 功能（24px）：功能卡片、图片、面板
- 区块（40px）：大区块、页脚、容器
- 药丸形（1584px）：CTA、药丸形按钮

## 6. 深度与层级

| 级别 | 处理方式 | 用途 |
|------|----------|------|
| 扁平（Level 0） | 无阴影，奶油色画布 | 页面背景 |
| Clay Shadow（Level 1） | `rgba(0,0,0,0.1) 0px 1px 1px, rgba(0,0,0,0.04) 0px -1px inset, rgba(0,0,0,0.05) 0px -0.5px` | 卡片、按钮——带内嵌高光的多层阴影 |
| Hover Hard（Level 2） | `rgb(0,0,0) -7px 7px` | 悬停状态——有趣的硬边偏移阴影 |
| Focus（Level 3） | `rgb(20, 110, 245) solid 2px` | 键盘焦点环 |

**阴影哲学**：Clay 的阴影系统独特地分为三层：向下投射（`0px 1px 1px`）、向上内嵌高光（`0px -1px 1px inset`）和微妙边缘（`0px -0.5px 1px`）。这创造了"压入粘土"的质感，元素既感觉凸起又嵌入——就像内容被压印在表面的粘土板。悬停硬阴影（`-7px 7px`）刻意采用复古图形风格，参考印刷时代的投影，增添物理趣味。

### 装饰性深度
- 全宽色板颜色区块通过色彩对比创造戏剧性深度
- 虚线边框与实线边框并存，增添视觉纹理
- 产品插图采用温暖、有机的艺术风格

## 7. 设计准则

### 应该做的
- 使用温暖奶油色（`#faf9f7`）作为页面背景——温暖感是身份标识
- 在 Roobert 标题上应用全部 5 组 OpenType 样式集：`"ss01", "ss03", "ss10", "ss11", "ss12"`
- 使用命名色板调色板（Matcha、Slushie、Lemon、Ube、Pomegranate、Blueberry）作为区块背景
- 应用有趣的悬停动画：`rotateZ(-8deg)`、`translateY(-80%)`、硬阴影 `-7px 7px`
- 使用温暖燕麦色边框（`#dad4c8`）——而非中性灰
- 混合实线和虚线边框以增加视觉多样性
- 使用大圆角：卡片 24px，区块 40px
- 标题专用字重 600，UI 用 500，正文用 400

### 不应该做的
- 不要使用冷灰背景——温暖奶油色（`#faf9f7`）不可妥协
- 不要使用中性灰边框（`#ccc`、`#ddd`）——始终使用温暖燕麦色调
- 不要在同一区块混合超过 2 种色板颜色
- 不要跳过 OpenType 样式集——它们定义了 Roobert 的个性
- 不要使用微妙的悬停效果——旋转 + 硬阴影是标志性交互
- 不要在功能卡片上使用小圆角（<12px）——大圆角是结构性的
- 不要使用标准阴影（模糊型）——Clay 使用硬边偏移和多层内嵌阴影
- 不要忘记带 1.08px 字间距的大写标签——它们是导视系统

## 8. 响应式行为

### 断点
| 名称 | 宽度 | 关键变化 |
|------|------|----------|
| 小屏移动端 | <479px | 单列，紧凑内边距 |
| 移动端 | 479–767px | 标准移动端，堆叠布局 |
| 平板 | 768–991px | 2 列网格，精简导航 |
| 桌面端 | 992px+ | 完整布局，3 列网格，扩展区块 |

### 触摸目标
- 按钮：最小 6.4px + 12.8px 内边距以确保足够的触摸区域
- 导航链接：15px 字体配大间距
- 移动端：全宽按钮便于点击

### 折叠策略
- 主视觉区：80px → 60px → 更小的展示文本
- 导航：水平布局 → 767px 处折叠为汉堡菜单
- 功能区块：多列 → 堆叠
- 彩色区块：保持全宽但压缩内边距
- 卡片网格：3 列 → 2 列 → 单列

### 图片行为
- 产品截图按比例缩放
- 彩色区块插图适应视口宽度
- 圆角在各断点保持一致

### 迭代指南
1. 从温暖奶油色（#faf9f7）开始——绝不用冷白色
2. 色板颜色用于全宽区块，而非小面积强调——大胆使用 matcha、slushie、ube
3. 到处使用燕麦色边框（#dad4c8）——虚线变体用于装饰
4. OpenType 样式集是必需的——它们让 Roobert 看起来像 Roobert
5. 悬停动画是标志性特征——旋转 + 硬阴影，而非微妙渐变
6. 大圆角：24px 卡片，40px 区块——没有任何东西看起来尖锐或企业化
7. 三种字重：600（标题）、500（UI）、400（正文）——严格的角色分工
